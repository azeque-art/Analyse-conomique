# Bénin vs CEDEAO : dix ans d'évolution économique (2015-2025)

Depuis 2015, le Bénin a engagé plusieurs réformes économiques : 
modernisation du port de Cotonou, digitalisation de la fiscalité, 
création de zones industrielles. Ce projet cherche à savoir si ces 
efforts se traduisent réellement dans les chiffres, en comparant le 
Bénin à cinq voisins de la CEDEAO sur la décennie 2015-2025.

## La question de départ

Le Bénin rattrape-t-il ses voisins, stagne-t-il, ou perd-il du terrain ?

## Les pays comparés

Bénin (le pays de référence), Nigeria, Ghana, Côte d'Ivoire, Sénégal 
et Togo.

## Ce qu'on regarde

- PIB et PIB par habitant
- Inflation
- Investissements directs étrangers (IDE)
- Commerce extérieur
- Dette publique

## D'où viennent les données

Banque mondiale, FMI, BCEAO, et l'INSAE côté béninois. Le détail des 
sources et des liens de téléchargement est dans `data/SOURCES.md`.

## La méthode

Le détail de la méthodologie (pourquoi ces indicateurs, quelles 
méthodes statistiques, quelles limites) est expliqué dans 
`docs/METHODOLOGIE.md`.

## Organisation du dépôt

- `data/raw` : données brutes, telles que téléchargées
- `data/processed` : données nettoyées
- `notebooks` : exploration et analyse
- `src` : scripts réutilisables
- `reports` : résultats et figures
- `docs` : documentation
