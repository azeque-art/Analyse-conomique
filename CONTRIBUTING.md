# Comment contribuer

Merci de t'intéresser à ce projet. Toute aide est bienvenue, que tu 
sois économiste, développeur, statisticien, ou juste débutant curieux.

## Ce que tu peux faire

- Collecter ou vérifier des données (voir `data/SOURCES.md`)
- Nettoyer des jeux de données
- Écrire ou améliorer des scripts d'analyse
- Relire la méthodologie et proposer des corrections
- Améliorer la documentation

Regarde l'onglet **Issues** du dépôt pour voir les tâches déjà 
identifiées, en particulier celles marquées `good first issue` si tu 
débutes.

## Si tu ne connais pas Git/GitHub

Tu peux modifier les fichiers directement depuis le site, sans rien 
installer :

1. Ouvre le fichier que tu veux modifier
2. Appuie sur l'icône crayon (modifier)
3. Fais tes changements
4. Descends en bas, décris brièvement ce que tu as fait
5. Valide avec "Commit changes" (ou "Propose changes" si tu n'as pas 
   les droits d'écriture directs, ça créera une pull request)

## Si tu connais Git

```bash
git clone https://github.com/azeque-art/Analyse-economique.git
cd Analyse-economique
git checkout -b nom-de-ta-branche
# tes modifications
git add .
git commit -m "Description claire de ce que tu as fait"
git push origin nom-de-ta-branche
